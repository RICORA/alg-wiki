# プログラムを書く環境を整える

## プログラムを最低限書けるようにしよう

- VS Codeを入れる

- (Mac) Xcode Command Line Tools (CLT)を入れる

## パッケージマネージャーでソフトを管理しよう

- Windows

   - Chocolateyを入れよう
   - wingetとかscoopはそのうち...

- Mac

   - homebrewを入れよう

- Linux

   割愛。`apt`なり`pacman`を使っているはずなので自分で調べてね

## gitが使えるようにしよう

- Windows

   - Git Bashを入れよう

- Mac

   - brewでgitを入れよう

- Linux

   - マネージャで入れる。問題が生じたらArch wikiかdebianのメーリスとか見る。
